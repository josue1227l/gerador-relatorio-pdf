# Gerador Automático de Relatórios em PDF

Este projeto gera automaticamente um relatório em PDF a partir de dados em um arquivo CSV.

## Objetivo
Automatizar a criação de relatórios, evitando processos manuais.

## Tecnologias utilizadas
- Python
- Pandas
- ReportLab
